# Sentiment Analysis Scripts

This folder contains the scripts and output logs for the sentiment analysis performed on the US-China Trade War Corpus.

## Files

| File | Description |
|------|-------------|
| run_sentiment_analysis.py | Python script to run Pysentimiento and VADER sentiment analysis |
| sentiment_analysis_output.txt | Terminal output log from running the analysis |

## Requirements

pip install pandas numpy scipy pysentimiento vaderSentiment

## Usage

python run_sentiment_analysis.py
